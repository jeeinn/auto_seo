# seo_urls
帮你免费生成网站外链地址。Free site outside the chain urls.

## 说明
源码非本人作品，来源不明，属于二次开发，侵删！

## 食用方法
请直接修改 `config.php` 文件，即可配置全站内容。

## 外链网址扩展
* `urls.txt`内容为收录外链页面地址
* 一行代表一个可收录外链的网址
* 网址中 `***` 代表可替换待优化的域名

**如果你有更好的网址，建议提交**
